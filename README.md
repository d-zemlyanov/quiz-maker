# QuizMaker — консольное приложение для создания и прохождения викторин

## Функциональные возможности

### Пользователи и роли
- Регистрация и авторизация (логин/пароль).
- Две роли: **Администратор** и **Пользователь**.
- Администратор имеет полный доступ к управлению викторинами.

### Управление викторинами (только админ)
- Создание новой викторины: название, описание, вопросы, варианты ответов.
- Редактирование существующей викторины: добавление/удаление вопросов, изменение ответов.
- Удаление викторины.

### Прохождение викторин (все пользователи)
- Выбор викторины из списка.
- Поочередный вывод вопросов с вариантами ответов.
- Подсчёт результата (количество правильных ответов, процент).
- Сохранение результата в статистику.

### Статистика
- Просмотр своей статистики (дата, викторина, результат).
- Администратор может просматривать статистику по всем пользователям и викторинам.

### Хранение данных
- Все данные хранятся в JSON-файлах.
- При первом запуске создаются необходимые директории и файлы.
